Research

Scientific Foundation Models (SciFM)

My research in Scientific Foundation Models focuses on developing foundational AI models specifically designed for scientific applications. This work involves creating large-scale models that can understand and generate scientific content, solve complex mathematical problems, and assist in scientific discovery. Key areas include:

  • Development of pre-trained models for scientific data
  • Integration of physical constraints in neural architectures
  • Transfer learning for scientific applications

Scientific Machine Learning (SciML)

In SciML, my work focuses on combining traditional scientific computing methods with modern machine learning approaches. This includes:

  • Neural networks for solving partial differential equations
  • Physics-informed machine learning models
  • Data-driven discovery of physical systems
  • Optimization techniques for scientific computing

Large Language Models (LLM)

My research in LLMs focuses on their application in scientific domains. This includes:

  • Scientific text understanding and generation
  • Code generation for scientific computing
  • Mathematical reasoning and problem-solving
  • Integration of domain knowledge in language models

3D Geometry Problems

In the field of 3D geometry, my research focuses on:

  • Deep learning for 3D shape analysis
  • Geometric deep learning
  • Point cloud processing and analysis
  • 3D reconstruction and understanding
